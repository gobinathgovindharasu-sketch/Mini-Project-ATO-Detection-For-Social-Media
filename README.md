# Mini-Project-ATO-Detection-For-Social-Media

Real-time Social Media Account Takeover (ATO) Detection Tool using ensemble ML (Isolation Forest + One-Class SVM), Fernet encryption, and a dark cyber-themed UI.

See the full project in [`ato-detection-tool/`](./ato-detection-tool/) with complete documentation in [`ato-detection-tool/README.md`](./ato-detection-tool/README.md).

## Quick Start

```bash
cd ato-detection-tool
cp .env.example .env
docker-compose up --build
```

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
